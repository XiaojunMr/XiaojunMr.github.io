css特殊盒模型

2016-11-23

css怪异盒模型

<!doctype html>
<html>
<head>
    <meta charset=" utf-8"/>
    <title>怪异盒模型</title>
    <style type="text/css">
        *{
        margin:0px;
        padding: 0px;

        }
        .box{
            /*
            可以设置一个元素的box-sizing：border-box；
            让他成为一个怪异的盒模型，
            怪异盒模型的padding、border不在影响元素本身的大小。
            */
            box-sizing: border-box;
            display: -webkit-box;
            width: 200px;
            height: 200px;
            background-color: gold;
            border: 20px solid pink;
            /*padding-left:30px; */



        }
    </style>
</head>
<body>
    <div class="box">123</div>
</body>
</html>

css弹性盒模型

<!doctype html>
<html>
<head>
    <meta charset="utf-8"/>
    <title>弹性盒模型</title>
    <style type="text/css">
        .box{
            /*-webkit-box
                设置元素为弹性盒模型。弹性盒模型可以按比例划分区域，让子元素去分配
            */
            display: -webkit-box;
            width: 200px;
            height: 200px;
            background-color: gray;
            border: 5px solid gold;
            /*划分的顺序
                默认在前面的子元素线分配，可以设置-webkit-box-direction为reverse 反向分配
            */
            -webkit-box-direction:reverse;
            /*-webkit-box-orient:
                可以设置横向或者纵向划分     vertical 代表纵向 上到下划分，默认横向
            */
            -webkit-box-orient:vertical;
        }
        .box .div1{
            -webkit-box-flex:1;
            background-color: black;
        }
        .box .div2{
            -webkit-box-flex:2;
            background-color: blue;
        }
        .box .div3{
            -webkit-box-flex:3;
            background-color: purple;
        }
        p{
            -webkit-column-count:10;
            -webkit-column-rule:1px solid pink;
            -webkit-column-gap:10px;
        }
    </style>
</head>
<body>
    <div class="box">
        <div class="div1"></div>
        <div class="div2"></div>
        <div class="div3"></div>

    </div>    
    <p> 我是谁123  我是谁123  我是谁123  我是谁123

    我是谁123  我是谁123  我是谁123  我是谁123  
    我是谁123  我是谁123  
    </p>
</body>
</html>
